# Drips Readiness - ValidTrust SDK

This document outlines the Drips readiness for the ValidTrust SDK repository.

## Repository Purpose

validtrust-sdk is the official TypeScript developer toolkit for the ValidTrust Network. It abstracts the complexity of Soroban RPC calls, transaction building, and wallet integration.
