# Contributing to Validity SDK

We welcome contributions to the Validity SDK! Follow these guidelines to get started.

## Setup

1. Fork the repo and clone it locally.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the project:
   ```bash
   npm run build
   ```
4. Run tests:
   ```bash
   npm test
   ```

## Submitting Pull Requests

- Keep your code clean and well-documented.
- Write Jest tests for any new features.
- Update `docs/` and `README.md` if the public API changes.
- Submit a PR to the `main` branch.
